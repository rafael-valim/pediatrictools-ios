Pediatric Tools

Clinical Calculators for Pediatrics

About the App

Pediatric Tools is a comprehensive suite of 20+ evidence-based clinical calculators designed for pediatric healthcare professionals. From Apgar scores to Phoenix Sepsis criteria, all tools are available offline with no account required.

Key Features

Evidence-based scoring tools, drug dosage calculators, growth percentile charts, and critical care assessments. Available in English, Portuguese, Spanish, and French. All calculations run locally on your device with no data collection.
